# ShopQuanAo - Android Shopping App

## Introduction

ShopQuanAo is an Android e-commerce application developed using Java and PHP/MySQL. The application allows users to browse products, add products to cart, place orders, and manage products through an admin panel.

## Features

### User

* Login / Register
* View banners and product categories
* View product details
* Add products to cart
* Update cart quantity
* Remove products from cart
* Checkout and create orders
* Navigation Drawer Menu
* Settings page

### Admin

* Add products
* Edit products
* Delete products
* Manage product information
